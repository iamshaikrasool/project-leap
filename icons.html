App Icons

We already have a lot of icons and a handy rails helper in the Treehouse app, here's how to use it.

Displaying an icon:

  • Locate the unique id of the icon, which is the filename of the icon itself. (e.g. #logo)
  • In most cases, using the icon helper with the unique id will work just fine. (e.g. <%= icon('#logo') %>)
  • In some cases, you may need to add styles/color to several paths inside of the icon. In this case, you'll need inject the svg/html of the icon directly into the page by using the path to the icon file. (e.g. <%= icon('layouts/treehouse/ipad') %>)

Styling an icon:

  • To change the color of an icon, simply apply the following css: .logo-icon { fill: #fff }
  • To add a drop-shadow to the icon, simply add: .logo-icon { box-shadow: 0 1px 0 rgba(#000,.5) }
  • To change the size of an icon, simply add: .logo-icon { width: 100px; height: 100px }
