Typography

Headers

Headers are built using typical html elements h1 - h6. To give ourselves a bit more control, we've created utility classes for applying different sizes to the default elements so that we can control markup hierarchy while still applying the sizes we want visually (.h1-xs, .h2-xs, .h3-xs, .h4-xs, .h5-xs, .h6-xs). By default, headers have no margin and padding applied to them.
